**Data Analyst Internship Portfolio - Gaurav Singh Shekhawat**

Welcome to my Data Analyst Internship repository! This workspace contains the code, queries, dashboards, and documentation for the projects I developed during my internship.

This repository serves as a comprehensive log of the projects, methodologies, and business intelligence dashboards developed during my remote Data Analyst Internship at Elevate Labs.

The program focused on practical, hands-on experience across the entire data analysis lifecycle: from raw data ingestion and systematic cleaning to data modeling and interactive visualization. The core objective was to generate contextual business insights by integrating industry-standard tools.

**CORE TECHNOLOGY STACK:**

**Microsoft Excel:** Advanced Formulas (IF, VLOOKUP), Power Query for data transformation, Pivot Tables, Pivot Charts, and Slicers.
**SQL (Structured Query Language):** Data extraction from relational databases using complex queries, GROUP BY, and JOINs (INNER/LEFT).
**Microsoft Power BI:** Power Query Editor for ETL processes, Star Schema data modeling, DAX (Data Analysis Expressions) for custom KPIs, and interactive dashboard design.

**PROJECT TIMELINE & KEY INSIGHTS:**

The internship consisted of six progressively complex projects, culminating in an end-to-end capstone analysis.

**Week 1: Bike Sales Analysis (Excel)**

**Objective:** Clean a raw dataset and build a preliminary dashboard to identify key customer segments.
**Process:** Handled missing values, standardized string inconsistencies, and created dynamic age brackets using nested IF statements.
**Insight:** The 'Middle Age' demographic (40-60 years old) with a '0-1 Miles' commute represented the most valuable and frequent purchasing segment.

**Objective:** Master granular data segmentation using advanced Pivot Tables.
**Process:** Built distinct visualizations analyzing the intersection of Job Industry, State, and Wealth Segment.

**Week 3: Store Annual Report (Excel Dashboarding)**

**Objective:** Aggregate 12 months of e-commerce data into a single executive dashboard.
**Process:** Consolidated time-series data and developed KPIs tracking Order Status, Demographics, and Sales Channels, all connected via interactive Slicers.
**Insight:** Maharashtra was the top-performing state generating 2.99M in sales, with Amazon acting as the dominant channel driving 35% of all orders.

**Objective:** Transition to Power BI to analyze a large-scale global streaming dataset.

**Insight**: Demonstrated platform capability by visualizing massive data scales, highlighting The Weeknd's 'Blinding Lights' as the top track with 4 billion streams.

**Objective:** Perform time-series analysis on historical financial market data.

**Week 6: Bike Shop Capstone Project (SQL + Excel + Power BI)**

**Objective: **Execute an end-to-end analytical workflow simulating real-world data team tasks.
**Process:** Extracted and joined multiple relational tables (Riders, Trips, Payments) using SQL, validated the anomalies in Excel, and built a comprehensive Star Schema model in Power BI.
**Insight:** Identified that loyal 'registered' users drive 81.17% of the business. Discovered that Season 3 (Summer) is peak profitability ($4.9M), with distinct hourly usage spikes at 8 AM and 5-6 PM, indicating a strong work-commute customer base.
